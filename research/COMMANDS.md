# `ch --help`

LLM friendly help.

# `chimera` - long form alias

# `ch register {path}`

Register a project into Lycia.

# `ch {project} start {goal} {source}`

Start working on a goal, the source may be a Linear issue.
If it's not specified, interact with the user to establish it and then document it in a bead.

process: create a worktree for the goal

# `ch init {path}`

Create new Lycia to work in.

# `ch doctor [--fix]`

Check the health of your Lycia.

# `ch status [--project {project}] [--goal {goal}]`

Show the status of your Lycia.
